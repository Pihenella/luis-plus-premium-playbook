# Luis+ Premium Playbook

Статический калькулятор премии по логике Excel-файла `Калькулятор дохода 01_2026_МП_корр.xlsx`.

## Что внутри

- `index.html` — интерфейс калькулятора.
- `styles.css` — визуальный слой в стиле рабочего дашборда Luis+.
- `app.js` — формулы, шкалы KPI и предзаполненные данные за январь–апрель 2026 года.

## Как открыть локально

Открой файл `index.html` в браузере.

## Как включить GitHub Pages

1. В настройках репозитория открой `Pages`.
2. Выбери ветку `main` и папку `/root`.
3. Сохрани настройки.

Если репозиторий приватный, доступность Pages зависит от настроек аккаунта GitHub.

## Важное по расчёту

В Excel база премии считается как:

```text
Деньги × Рентабельность 6М
```

На скриншотах есть факт по Д/С, валовой прибыли и обороту, но нет отдельного поля «Рентабельность 6М». Поэтому в калькуляторе это поле вынесено в таблицу по месяцам. По умолчанию стоит `20%`, как в исходном Excel-файле.
